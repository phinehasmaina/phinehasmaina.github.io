Color coded delivery

This was my second project for the second semester of my freshman year! For this project my team and I designed a robot using the lego Mindstorms kit and arduino boards. The main objective of our robot was to deliver Color coded packages to the corresponding color coded house. The Titanic

This was my very first successful python project! In this python project, my team and I used several python algorithms to try make a machine learning model that could predict if one would survive the titanic accident. We used the catboost algorithm to make our model and that had an accuracy of 80%. This model was posted on kaggle where our model ranked top 10% of all models that have been submitted to the competition!
